🎮 메인 프로젝트: AR 캐릭터 × LLM 대화 앱 — 갤노트20에서 동작하는 온디바이스 AI 캐릭터. Week 11(8월)부터 시작해서 Week 32(12월말)까지 22주간 진행합니다.

📱 프로젝트 개요

갤럭시 노트20 울트라 · Unity AR Foundation · 온디바이스 LLM · TTS

카메라를 켜면 평면 위에 3D 캐릭터가 등장하고, 사용자 음성을 인식해서 캐릭터가 자연스럽게 대화하는 모바일 AR 앱. 클라우드 의존 없이 폰 안에서 LLM 추론까지 모두 처리.

🎯 핵심 차별화 포인트 (자소서/면접용)

  • 온디바이스 LLM — 1~2B 파라미터 모델을 INT4 양자화로 폰에서 실행
  • AR 통합 — Unity AR Foundation으로 ARCore 활용
  • 실시간 대화 — STT → LLM → TTS 파이프라인 5초 이내
  • 프라이버시 보존 — 모든 처리가 디바이스 내부에서
  • 한국어 지원 — 한국어 fine-tuned 모델 활용

🛠️ 기술 스택

AR/3D: Unity 2022 LTS, AR Foundation, ARCore
LLM: Gemma 2B / Qwen 2.5 1.5B, llama.cpp Android
음성: Whisper Tiny (STT), Android TTS
경량화: GGUF 4-bit 양자화, ONNX
캐릭터: VRM 또는 Ready Player Me
개발: Android Studio, Git LFS

📅 22주 마일스톤

Week 11-14 (8월) 기획 + 베이스라인 (Unity AR 캐릭터 표시)
Week 15-18 (9월) 하반기 인턴 지원 + LLM 모바일 통합
Week 19-22 (10월) 음성 파이프라인 (STT/TTS) 통합
Week 23-26 (11월) 최적화 + 캐릭터 애니메이션
Week 27-32 (12월) UX + 문서화 + 데모 영상 + 블로그 시리즈

⚠️ 갤노트20에서 예상되는 도전

  • Snapdragon 865 + 12GB RAM — LLM 추론은 가능하지만 토큰 속도 느림
  • 발열 → 추론 시간 제한 필요 (대화당 토큰 수 제한)
